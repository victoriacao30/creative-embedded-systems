# Sonder — Generative Art Installation (ESP32 TTGO T-Display)

## Overview
This project is a generative art installation created using an ESP32 TTGO T-Display inspired by the concept of Sonder— the realization that every passerby is living a life as vivid and complex as one’s own. The display presents an evolving cityscape visualization followed by contextual text, creating a continuously changing visual experience rather than a fixed animation.

The work was designed for a battery-powered hallway installation in Milstein Hall as part of a collaborative class exhibition.


## Hardware Requirements
- ESP32 TTGO T-Display  
- 3.7V LiPo battery  
- USB-C cable for programming  


## Software Requirements
- VS Code  
- PlatformIO Extension  
- Arduino Framework  
- TFT_eSPI library
